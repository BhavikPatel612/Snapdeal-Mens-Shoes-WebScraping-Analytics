# Snapdeal-Mens-Shoes-WebScraping-Analytics 
**STAR Method Explanation**

**S – Situation**
- E-commerce websites contain large amounts of product data, making it difficult to manually compare product prices, discounts, ratings, and brand performance.”

**T – Task**
- My task was to perform web scraping on Snapdeal Men’s Sports Shoes data, analyze product insights, and build an interactive analytics dashboard for business analysis.

**A – Action**

1. Web Scraping
- Used Python for web scraping.
- Extracted product information from Snapdeal website such as:
Product Name,Brand,Original Price,Discount Percentage,Ratings,Reviews
 - Libraries Used
BeautifulSoup
Requests
Pandas

2. Data Cleaning & Processing
- Cleaned scraped data using Pandas.
- Removed missing and duplicate values.
- Converted price and discount columns into numerical format.
- Prepared structured dataset for dashboard analysis.

3. Data Analysis

- Performed product analysis such as:
- Brand-wise Product Count
- Average Discount Percentage
- Product Rating Analysis
- Original Price Comparison
- Review Analysis

**Created an interactive analytics dashboard using Power BI.**
4. Dashboard Development Charts Usage & Insights
- KPI Cards → displayed Total Products, Average Original Price, Average Discount %, and Average Rating.
- Bar Chart → compared total products by brand.
- Donut Chart → visualized average discount percentage by brand.
- Scatter Plot → analyzed relationship between ratings, reviews, and discount percentages.
- Clustered Column Chart → compared average discount percentage and average original price by product.
- Slicers → enabled interactive filtering based on brand, rating, and discount percentage.

6. Dashboard Interactivity
Added filters and slicers for dynamic product analysis.
Improved dashboard usability through interactive visuals and comparisons.

**R – Result**
- The dashboard successfully provided product pricing, discount, and rating insights for Snapdeal Men’s Sports Shoes.
- The project helped analyze brand performance, customer ratings, and discount trends using Web Scraping, Python, and Power BI.
